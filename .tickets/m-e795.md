# Use GetRequiredService for JsonSerializerOptions in handlers

All three handlers use GetService with an unsafe downcast to resolve JsonSerializerOptions, which silently returns null if not registered. Switch to GetRequiredService for immediate failure. Also consider deduplicating by injecting JsonSerializerOptions as a handler parameter.
