# Use IHttpClientFactory instead of manual HttpClient

Program.fs creates HttpClient directly with 'new HttpClient()' which is a .NET anti-pattern. Doesn't respect DNS TTL changes and risks socket exhaustion. Should register IHttpClientFactory in DI and inject it.
